Chainvayler demo application

Chainvayler is a new and innovative way of persisting and replicating POJO (Plain Old Java Object) graphs transparently.

This sample application makes use of Chainvayler's persistence capabilities.

This application uses neither SQLite, nor Room, nor any DAO or SharedPreferences, its data objects are automagically and transparently persisted!
